# TodoApp
Apliacion paga gestionar tareas.

## instalacion 
1- En la raiz crea un archivo .env con las siguientes variables.
PORT = 3000
DB_USER = postgres
DB_PASSWORD = ''
DB_NAME = todoApp
JWT_SECRET = ''

2- En postgre crea una base de datos llamada todoApp

3- npm run install-dependencies

4- npm run db

5- npm start
